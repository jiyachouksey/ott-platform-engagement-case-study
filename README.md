# OTT Platform User Engagement & Retention Case Study

**Author:** Jiya Chouksey  
**Tools:** Python (Pandas, Matplotlib, Seaborn), Jupyter Notebook  
**Dataset:** Kaggle - OTT Content Scheduling for Engagement  

##  Overview
This case study analyzes user behavior and engagement patterns on a streaming platform to determine the best content scheduling and promotion strategies.

## Key Insights
- Peak engagement between **7 PM – 10 PM**.
- **Comedy** and **Thriller** genres show highest retention.
- **18–35 age group** dominates viewing hours.
- Promotional content drives **1.5× higher engagement**.

##  Recommendations
- Release new content during prime engagement hours.
- Prioritize popular genres in marketing campaigns.
- Focus on younger audiences for higher retention.
